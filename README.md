# 💧 Quién quiere detectar UN MILLÓN de fugas

![Version](https://img.shields.io/badge/version-2.0-gold)
![License](https://img.shields.io/badge/license-Apache%202.0-blue)
![Status](https://img.shields.io/badge/status-active-green)

Juego interactivo estilo "Quién quiere ser millonario" diseñado para concienciar sobre la detección de fugas de agua. Sistema multiplayer en tiempo real con Google Apps Script como backend.

---

## 🎮 Características

- ✅ **Sistema multiplayer en tiempo real** con sincronización de estado
- 🎯 **3 interfaces independientes**: Jugador (móvil), Pantalla (TV/proyector), Admin (moderador)
- 🏆 **Sistema de puntuación avanzado**: 5 puntos al más rápido, -0.2 pts por segundo de retraso
- 🎵 **Audio dinámico**: Música de fondo, efectos de sonido y soundboard manual
- 🃏 **Comodines clásicos**: 50%, Público y Llamada
- 📊 **Ranking en tiempo real** con persistencia de sesiones
- 🎨 **Diseño glassmorphism** con animaciones fluidas
- 📱 **Responsive** y optimizado para móviles
